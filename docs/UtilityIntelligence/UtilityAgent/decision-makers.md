title: Decision Makers
---

In **Utility Intelligence**, each agent contains a list of decision makers and each decision maker contains a list of decisions. Here's how the decision-making process of a utility agent works step by step:
1. For each decision maker, the agent calculates the score of all of its decisions and selects the best decision. 
2. After that, the agent compares the scores of the best decisions from each decision maker with each other, and the winner is the decision with the highest score.
