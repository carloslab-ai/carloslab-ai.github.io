# How do Decision Makers work?

In **Utility Intelligence**, each agent contains a list of decision makers, and each decision maker contains a list of decisions. And the goal of each agent is to make the best decision based on the current situation.
Here's how the decision-making process of a utility agent works:
1. For each decision maker, the agent calculates the score of all of its decisions and selects the best decision. 
2. Afterwards, the agent compares the scores of the best decisions from each decision maker with each other, and the winner is the decision with the highest score.

# Creating a new Decision Maker

To create a new Decision Maker, you need to go to the **Agent Tab**, give the new Decision Maker a name, and then click the **Create** button:
